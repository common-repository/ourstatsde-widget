create a widget for the ourstats.de counter service

== Description ==

ourSTATS.de is a free Service to find out information about your visitors such as eg Screen resolution, browser, language and origin, page views by hour survey, as well as what keywords your visitors use to go to your homepage.

This plugin creates a widget that displays the ourstats.de counter code on your blog. You can always change the button style counter.
Look at: http://www.ourstats.de/

== Installation ==

This section describes how to install the plugin and get it working.

e.g.

1. Upload the `ourstatsde-widget` folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Drag&Drop the widget in your sidebar
